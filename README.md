# football-ai-project

Football AI project with comprehensive data analysis and services.

## Project Structure

- `football_ai_service/` - AI service implementation
- `base_data/` - Base football data
- `commentary_data/` - Match commentary data
- `keyEvents_data/` - Key match events
- `lineup_data/` - Team lineup information
- `playerStats_data/` - Player statistics
- `plays_data/` - Play-by-play data
- `logs/` - Application logs
